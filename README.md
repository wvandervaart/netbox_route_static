# Requirements

* Netbox 3.7+
* Python 3.10+

## Compatibility Matrix

|       | Netbox 3.7.x | 
|-------|--------------|
| 1.0.0 | X            |

## Installation

To install, simply include this plugin in the plugins configuration section of netbox.

Example:
```python
    PLUGINS = [
        'netbox_route_static'
    ],
```
